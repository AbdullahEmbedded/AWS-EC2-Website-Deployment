# ☁️ AWS EC2 Website Deployment

This project demonstrates the deployment of a static e-commerce website on an Amazon EC2 instance using **Apache HTTP Server**. It covers the complete deployment process, from launching an EC2 instance to making the website publicly accessible.

---

## 🚀 Project Overview

The main objective of this project was to learn cloud deployment using Amazon Web Services (AWS). The website was hosted on an Amazon Linux EC2 instance and configured using Apache Web Server.
