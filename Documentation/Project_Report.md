# AWS EC2 Website Deployment Project Report

## Project Title

Deploying a Static Website on Amazon EC2 using Apache HTTP Server

---

# Objective

The objective of this project was to learn how to deploy a static website on a cloud server using Amazon Web Services (AWS). The project includes launching an EC2 instance, connecting through SSH, installing Apache, configuring the server, and making the website publicly accessible.

---

# Technologies Used

- Amazon Web Services (AWS)
- Amazon EC2
- Amazon Linux
- Apache HTTP Server
- Linux Terminal
- SSH
- GitHub

---

# Project Workflow

1. Create an AWS account.
2. Launch an Amazon EC2 instance.
3. Generate and download a key pair.
4. Configure the security group to allow HTTP and SSH traffic.
5. Connect to the EC2 instance using SSH.
6. Update the operating system packages.
7. Install Apache HTTP Server.
8. Start and enable the Apache service.
9. Upload website files to the web server directory.
10. Verify the deployment by accessing the website through the EC2 public IP.

---

# Key Learning Outcomes

- Understanding AWS cloud services.
- Managing Amazon EC2 instances.
- Using Linux commands for server administration.
- Deploying a website with Apache.
- Configuring security groups.
- Connecting to remote servers using SSH.
